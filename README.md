                                               🍔 Food Munch - Restaurant Website

![Food Munch Banner](https://d1tgh8fmlzexmh.cloudfront.net/ccbp-responsive-website/foodmunch-banner-bg.png)

A responsive restaurant website built with HTML, CSS, and Bootstrap that showcases food menu, services, and online ordering features.

## 🌟 Features

- **Fully responsive** design for all devices
- **Modern UI** with Bootstrap components
- **Interactive sections**:
  - Menu gallery
  - Service highlights
  - Online payment options
  - Customer testimonials
- **Optimized performance** with CDN-hosted assets
